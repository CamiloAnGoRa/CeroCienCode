# 🚀 CeroCienCode

**De la lógica al despliegue: El camino integral para convertirte en desarrollador.**

---

## 📖 Sobre el Proyecto
Este es un aplicativo interactivo diseñado para guiar a futuros desarrolladores a través de un aprendizaje por **capas de abstracción**. A diferencia de otros cursos, aquí no solo aprendes a escribir código, sino a entender la arquitectura detrás de una aplicación real.

### 🏗️ Arquitectura del Aprendizaje (Las Capas)
1.  **Capa de Pensamiento:** Lógica pura, algoritmia y resolución de problemas.
2.  **Capa de Interfaz:** Estructura (HTML), Estilo (CSS) e Interactividad (JS/TS).
3.  **Capa de Datos:** Modelado, persistencia y gestión de bases de datos.
4.  **Capa de Negocio:** Lógica de servidor, APIs y seguridad (Back-end).
5.  **Capa de Integración:** Conexión total y despliegue en entornos productivos.

## 🛠️ Tecnologías Principales
*   **Frontend:** Next.js, React, Tailwind CSS.
*   **Backend:** Node.js, NestJS.
*   **Database:** PostgreSQL / MariaDB con Prisma ORM.
*   **Core:** WebAssembly para ejecución de código en tiempo real.

## 🌟 Características
- **Sandbox Interactivo:** Ejecuta código directamente en el navegador.
- **Feedback en Tiempo Real:** Explicaciones claras sobre errores de lógica.
- **Gamificación:** Sistema de progreso basado en hitos arquitectónicos.
